# Media Extractor 프로젝트 블루프린트

## 1. 프로젝트 개요
이 프로젝트는 YouTube, Instagram 등 다양한 소셜 미디어 플랫폼에서 동영상(MP4) 또는 음원(MP3)을 고화질로 추출할 수 있는 웹 기반 도구입니다. `yt-dlp` 라이브러리를 기반으로 하며, FastAPI 백엔드와 현대적인 Vanilla JS 프론트엔드로 구성되어 있습니다.

## 2. 구현된 기능 및 디자인

### 백엔드 (FastAPI)
- `yt-dlp`를 활용한 강력한 미디어 추출 로직 (`main.py`, `extractor.py`).
- POST `/api/download` 엔드포인트를 통한 URL 기반 다운로드 처리.
- `StaticFiles`를 이용한 프론트엔드 리소스(HTML, CSS, JS) 서빙.
- 오류 발생 시 명확한 HTTP 예외 처리.

### 프론트엔드 (Modern Web Standards)
- **HTML5/CSS3/Vanilla JS**: 프레임워크 없는 순수 웹 기술 사용.
- **반응형 디자인**: 모바일과 데스크톱 모두에서 최적화된 레이아웃.
- **UI 컴포넌트**: 토글 스위치(음원 전용 모드), 애니메이션 스피너, 결과 알림 카드.
- **시각적 요소**: `oklch` 기반의 색상 체계, 부드러운 그림자 효과, 그라데이션 배경 적용.

### 디자인 포인트
- **접근성**: 명확한 텍스트 대비와 시각적 피드백 제공.
- **상태 처리**: 다운로드 중 로딩 스피너를 통해 사용자에게 진행 상황 공유.
- **아이콘**: Font Awesome을 활용하여 직관적인 UI 구성.

## 3. 현재 작업 및 계획 (2026-03-06)

### 완료된 단계
1. `uvicorn` 서버 기반의 FastAPI 백엔드 구축.
2. `main.py` 정적 파일 서빙 및 API 엔드포인트 구현.
3. `index.html`, `style.css`, `main.js` 현대적 UI 구성 및 연동.
4. `.idx/dev.nix` 설정을 통해 클라우드 미리보기 환경에서 백엔드 기능 활성화 (uvicorn 연동).
5. 기존 정적 서버(`http.server`) 제거 및 포트 설정 최적화.

### 향후 계획
- 다중 플랫폼 URL 자동 감지 및 썸네일 미리보기 기능 추가.
- 다운로드 기록(히스토리) 브라우저 로컬 저장소에 보관.
- 오류 발생 시 더 상세한 가이드 메시지 노출.
- `ffmpeg`를 활용한 고해상도 비디오 병합 로직 최적화.
